Arquitectura Frontend y Comunicación API

DataTables — 3 Tipos

  • datatable.init.js — Pantallas principales: CRUD completo, export PDF/Excel, server-side
  • datatable.catalogs.init.js — Catálogos: CRUD modal, client-side, validación inline
  • datatable.partials.init.js — Parciales: detalle embebido en master-detail

SendHttpClient — Métodos

GET GetAsync<T> → Listar GET FindAsync<T> → Buscar POST PostAsync → Crear PUT PutAsync → Editar PUT DeleteAsync → Soft Delete GET ExistAsync → Duplicados
